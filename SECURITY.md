# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public GitHub issue
2. Email security@graycode.io directly
3. Include detailed reproduction steps
4. Allow time for acknowledgment before disclosure

## Scope

This policy covers:
- Hawk CLI
- Build system
- Plugin system
- API integrations

Out of scope:
- Third-party provider services
- User-developed plugins
- Local environment configurations

## Response Timeline

- Acknowledgment: 24-48 hours
- Initial assessment: 1-3 days
- Fix timeline: Based on severity

## Attribution

We credit reporters in the security advisory (with permission).
